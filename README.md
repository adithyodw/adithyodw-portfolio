# Adithyo Dewangga Wijaya — Portfolio

Personal portfolio website for [Adithyo Dewangga Wijaya](https://www.linkedin.com/in/adithyodewangga/) — Senior Solutions Manager at Singtel, Singapore.

## Stack

- **Framework**: Next.js 15 (App Router, static export)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI**: Radix UI + shadcn/ui
- **Font**: Onest (Google Fonts, display swap)
- **Deployment**: Vercel

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
